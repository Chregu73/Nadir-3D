BEWEGUNG & POSITIONIERUNG
G0 Eilgang G0 X[pos] Y[pos] Z[pos] F[S]
Bewegt den Kopf mit maximaler Geschwindigkeit. Nur für Leerfahrten ohne Drucken.
G0 X10 Y10 F3000 ; Schnelle Fahrt zum Start
WICHTIG: Nicht während der Extrusion nutzen!
G1 Linearfahrt G1 X.. Y.. Z.. E.. F..
Standardbefehl für Druckbewegungen. Bewegt Achsen koordiniert mit Materialfluss.
G1 X50 Y0 E2.0 F1200 ; Druckt eine 50mm Linie
G28 Homing G28 [X] [Y] [Z] Fährt Endstopps an, um den Nullpunkt zu kalibrieren.
G30 Z-Probe G30 Einmalige Bettmessung an aktueller Position per Sensor.
KOORDINATENSYSTEME
G90 Absolut G90 Alle Koordinaten beziehen sich auf den Maschinen-Nullpunkt.
G91 Relativ G91 Koordinaten beziehen sich auf die aktuelle Position.
G92 Reset Pos G92 E0 Setzt den internen Zähler (meist Extruder) auf Null zurück.
G21 Einheit mm G21 Stellt sicher, dass alle Maße als Millimeter interpretiert werden.
TEMPERATUREN
M104 Hotend Set M104 S[T] Heizt das Hotend im Hintergrund auf (kein Warten).
M109 Hotend Wait M109 S[T] Wartet, bis das Hotend die Zieltemperatur stabil erreicht hat.
M140 Bett Set M140 S[T] Heizt das Bett im Hintergrund auf.
M190 Bett Wait M190 S[T] Pausiert das Programm, bis das Bett warm genug ist.
HARDWARE & DIAGNOSE
M201 Max Beschl. M201 X.. Y.. Setzt die maximale Beschleunigung in mm/s². Hilft gegen Ringing.
M203 Max Speed M203 X.. Y.. Limitiert die absolute Höchstgeschwindigkeit pro Achse.
M119 Endstopps M119 Gibt den Status der Endschalter (offen/ausgelöst) in die Konsole aus.
M106 Lüfter an M106 S[0-255] Regelt Bauteillüfter-Geschwindigkeit (PWM).
M107 Lüfter aus M107 Schaltet den Bauteillüfter sofort ab.
G4 Pause G4 P[ms] Verweilen für eine definierte Zeit (Millisekunden).
